# 🎯 Mixed Training Guide - 3 Dataset Strategy

## Overview

This guide explains how to organize **3 different datasets** (images + videos) and train a single powerful model for **maximum accuracy**.

### Why 3 Datasets?

```
Dataset 1: Real images + Real videos
Dataset 2: AI-generated/Synthetic fake images
Dataset 3: Deepfake video frames
         ↓
    Combined training = 90%+ accuracy
```

---

## 📊 Dataset Strategy

### **Dataset 1: Authentic Content (Real)**
- ✓ Natural photos from various sources
- ✓ Real-world video recordings
- ✓ Cellphone videos

### **Dataset 2: AI-Generated Fakes**
- ✗ DALL-E/Midjourney generated images

### **Dataset 3: Deepfake Videos**

## 🔧 Data Organization

Your data should be organized as:

```
backend/data/
├── real/
│   ├── photo1.jpg           (real photo)
│   ├── photo2.png           (real photo)
│   ├── authentic_video1.mp4 (real video)
│   └── authentic_video2.avi (real video)
│
└── fake/
    ├── ai_image1.jpg        (AI-generated)
    ├── ai_image2.jpg        (deepfake)
    ├── fake_video1.mp4      (deepfake video)
    └── fake_video2.avi      (deepfake video)
```

**Important**: Each file MUST be correctly labeled (real/ or fake/)

## 🐛 Troubleshooting

### **Issue: "No training data found"**
```bash
# Solution: Check your data directory structure
ls -la backend/data/real/
ls -la backend/data/fake/

# Ensure files exist in both directories!
```

### **Issue: CUDA Out of Memory**
```bash
# Solution: Reduce batch size
python advanced_mixed_training.py --batch-size 16 --epochs 50
```

### **Issue: Training is very slow**
```bash
# Solution: Use GPU (check CUDA installation)
# Or reduce data size for testing
python advanced_mixed_training.py --local-only --epochs 20
```

### **Issue: Low accuracy**
```bash
# Solutions:
# 1. Add more balanced data
# 2. Train longer
python advanced_mixed_training.py --epochs 150

# 3. Use data augmentation
python advanced_mixed_training.py --augment --epochs 100

# 4. Check data quality (correct labels)
```
